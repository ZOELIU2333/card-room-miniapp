# Card Room Miniapp

熟人房优先的棋牌微信小程序项目资料仓。

## 当前定位

做一个可以和好友同玩、也能逐步支持自由匹配和积分体系的棋牌平台。第一版先聚焦微信小程序和熟人房体验：创建房间、邀请好友、输入房号、选择玩法、查看好友局状态。

充值系统暂不纳入第一版。

## 当前 UI 共识

核心不是普通棋牌大厅，而是一个简约、高级、深色调的好友牌局产品。底部需要固定一排主导航按钮。避免土味棋牌城、花哨手游感、红金国潮、强动画、廉价深绿金边、赌场感和生活类 App 感。
